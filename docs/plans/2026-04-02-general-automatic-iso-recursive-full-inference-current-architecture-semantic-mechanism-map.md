support claim.

  `TyMu` / `TMu`;

  same-lane retained-child route that can be named and audited; and

  output visible for the accepted clear-boundary packets.

What remains packet-specific debt:

- `sameLaneLocalRetainedChildTarget` is still one dedicated same-lane
  retained-child arm;
- `boundHasForallFrom` plus `not hasForall` are still admissibility filters,
  not a general polymorphism rule;
- `preserveRetainedChildAuthoritativeResult` is still a packet-local
  authoritative-preservation seam; and
- item `2` still has no search, ambiguity, or termination law explaining how
  these packet-specific reads become bounded general support.

That is why this mechanism map can explain current behavior without promoting
the repo to a broader readiness state.
