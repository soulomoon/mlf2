tie-breaking, or heuristic guessing. Under `N1`, those cases remain rejection

## Bounded Termination Discipline

The item-3 bounded termination discipline is an exact current-architecture
discipline inferred from the accepted mechanism map plus the read-only code
and test anchors named there.

1. Candidate generation is serial and finite because item `3` admits only two
   named route families plus one retained-child guard cluster:
   `rootNonLocalSchemeAliasBaseLike`,
   `sameLaneLocalRetainedChildTarget`, and
   `boundHasForallFrom` / `keepTargetFinal` / `targetC`.
2. `src/MLF/Elab/Run/ResultType/Fallback/Core.hs` keeps the search local to
   already named anchors such as `baseTarget`, `rootFinal`,
   `boundVarTargetRoot`, `schemeBodyTarget`, and the retained-child list
   already present in `cBindParents`. Item `3` authorizes no family-wide
   candidate enumeration beyond those anchors.
3. `boundHasForallFrom` and `rootFinalInvolvesMu` walk only finite bound,
   body, and arrow structure already present in the canonicalized constraint.
   `src/MLF/Constraint/Acyclicity.hs` remains read-only evidence that the
   representation stays acyclic rather than a license for cyclic search or
   multi-SCC search.
4. `src/MLF/Reify/Type.hs` remains read-only evidence that recursive shape is
   reified on the inherited iso-recursive representation without fallback
   widening. It does not authorize equi-recursive equality or ancestor-based
   rescue beyond the already named surfaces.
5. `src/MLF/Elab/TermClosure.hs` remains read-only evidence that
   authoritative retained-child output can be preserved when recursive
   structure already exists. It does not create a fresh candidate family, and
   it does not retry ambiguous cases until one happens to work.
6. `test/Research/SameLaneRetainedChildRepresentativeGapSpec.hs`,
   `test/Research/P5ClearBoundarySpec.hs`, and `test/PipelineSpec.hs` keep
   the accepted clear-boundary packet evidence and the reject-side nested
   `forall` contrast review-visible on the authoritative surfaces only.

Under this discipline, `N6` stays bounded because the candidate space never
grows past the two named families and the one retained-child guard cluster.
There is no lawful item-3 branch that retries fallback with a broader search,
walks cyclic graphs, widens to multi-SCC handling, or appeals to
equi-recursive equality when the named guards fail. If the named route arms
do not collapse the case to one lawful candidate, the case must reject
instead of expanding the search.
