## Honest Current Position

The current repository can honestly claim:

> general automatic iso-recursive inference across its representative family
> matrix inside the inherited explicit-only production boundary

That claim does **not** mean the repo can yet synthesize recursive types for
unannotated programs.

The inherited unannotated baseline remains:

- explicit recursive annotations are still the production admission source for
  recursive structure;
- recursive meaning stays iso-recursive;
- non-equi-recursive semantics stay mandatory;
- non-cyclic structural graph encoding stays mandatory; and
- no fallback, second interface, or silent widening is authorized.

## Unannotated Capability Contract

For this execution lane, “fully automatic unannotated iso-recursive
inference” means all of the following together on the existing authoritative
pipeline surfaces:

1. an unannotated program yields one reviewable inferred recursive root or
   bounded equivalent cluster without explicit source `μ`;
2. that inferred recursive result survives solver, elaboration,
   reconstruction, reification, and authoritative pipeline output;
3. ambiguity remains fail-closed rather than heuristic-ranked;
4. search remains bounded, explainable, and terminating; and
5. the result stays thesis-faithful inside the inherited semantic boundary,
   unless an explicit later architecture decision revises that boundary.

## Required Success Classes

A future positive claim must cover representative unannotated families for:

- local recursive-shape discovery;
- non-local propagation;
- owner-sensitive / retained-child placement;
- binder-sensitive placement;
- polymorphism / nested-`forall` interaction; and
- reconstruction-visible output.

## Required Fail-Closed Or Out-Of-Scope Classes

A future positive claim must still reject or explicitly classify:

- ambiguity requiring ranking;
- unsound owner / binder / scope crossings;
- equi-recursive-required cases;
- cyclic or multi-SCC-required cases;
- fallback- or second-interface-required cases; and
- open-ended termination pressure.
